Exercise 4. ( ★ )
Show that f:S → R is ρ -Lipschitz over an open convex set S if and only if for all w ∈ S and v ∈ ∂ f( w ) it is ∥ v ∥ ≤ ρ .
Hint:
You may use Cauchy-Schwarz inequality ⟨ y,x ⟩ ≤ ∥ y ∥ ∥ x ∥
 
Solution. Let f:S → R be ρ -Lipschitz over convex set S , w ∈ S and v ∈ ∂ f( w ) .
  • Since S is open we get that there exist ε >0 such as u:=w+ ε v ∥ v ∥ where u ∈ S . So ⟨ u-w,v ⟩ = ε ∥ v ∥ and ∥ u-w ∥ = ε .
  • From the subgradient definition we get f( u ) -f( w ) ≥ ⟨ u-w,v ⟩ = ε ∥ v ∥
  • From the Lipschitzness of f( ⋅ ) we get f( u ) -f( w ) ≥ ρ ∥ u-w ∥ = ρ ε
    Therefore ∥ v ∥ ≤ ρ .
    It is for all w ∈ S and v ∈ ∂ f( w ) it is ∥ v ∥ ≤ ρ .
  • For any u ∈ S , it is f( w ) -f( u ) ≤ ⟨ v,w-u ⟩ (  because  v ∈ ∂ f( w )   ) ≤ ∥ v ∥ ∥ w-u ∥ by Cauchy-Schwarz inequality (1) ≤ ρ ∥ w-u ∥ because  ∥ v ∥ ≤ ρ
  • Similarly it results u,w ∈ S f( w ) -f( u ) ≤ ⟨ v,u-w ⟩ ∥ v ∥ ≤ ∥ v ∥ ∥ u-w ∥ ≤ ρ ∥ u-w ∥ from (1) because w,u can be swaped in (1) as they both are any values in S .
