1 Motivations for stochastic gradient descent

Problem 1. Consider a learning problem ( H , Z , ℓ ) . Learning may involve the computation of the minimizer w * ∈ H , where H is a class of hypotheses, of the risk function (RF) R( w ) = E z ∼ g ( ℓ ( w,z ) ) given an unknown data generating model g( ⋅ ) and using a known tractable loss ℓ ( ⋅ , ⋅ ) ; that is w * = arg min ∀ w ∈ H ( R g ( w ) ) = arg min ∀ w ∈ H ( E z ∼ g ( ℓ ( w,z ) ) ) (1.1)

Remark 2. Gradient descent (GD) cannot be directly utilized to address Problem 1 (i.e., minimize the Risk function) because g is unknown, and because (1.1) involves an integral which may be computationally intractable. Instead it aims to minimize the ERF R ˆ ( h ) = 1 n ∑ i=1 n ℓ ( h, z i ) which ideally is used as a proxy when data size n is big (big-data).

Remark 3. The implementation of GD may be computationally impractical even in problems where we need to minimize an ERF R ˆ n ( w ) if we have big data ( n ≈ big), This is because GD requires the recursive computation of the exact gradient ∇ R ˆ n ( w ) = 1 n ∑ i=1 n ∇ ℓ ( w, z i ) using all the data { z i } at each iteration. That may be too slow.

Remark 4. Stochastic gradient descent (SGD) aims at solving (1.1), and overcoming the issues in Remarks 2 & 3 by using an unbiased estimate of the gradient (required by GD) based on a sample properly drawn from g .

2 Stochastic gradient descent

2.1 Description

Notation 5. For the sake of notation simplicity and generalization, we will present Stochastic Gradient Descent (SGD) in the following minimization problem w * = arg min ∀ w ∈ H ( f( w ) ) (2.1) where here f: R d → R , and w ∈ H ⊆ R d ; f( ⋅ ) is the unknown function to be minimized, e.g., f( ⋅ ) can be the risk function R g ( w ) = E z ∼ g ( ℓ ( w,z ) ) .

Algorithm 6. Stochastic Gradient Descent (SGD) with learning rate η t >0 for the solution of the minimization problem eq:gvbzvcb. where ∂ f( w ( t ) ) is the set of sub-gradients of f at w ( t ) .
For t=1,2,3,... iterate:
  1. compute w ( t+1 ) = w ( t ) - η t v t , (2.2) where v t is a random vector such that E ( v t | w ( t ) ) ∈ ∂ f( w ( t ) )
  2. terminate if a termination criterion is satisfied, e.g. If t ≥ T then STOP

Remark 7. If f is differentiable at w ( t ) , it is ∂ f( w ( t ) ) ={ ∇ f( w ( t ) ) } . Hence v t is such as E ( v t | w ( t ) ) = ∇ f( w ( t ) ) in Algorithm 6 step 1..

Note 8. Assume f is differentiable (for simplicity). We can re-write (2.2) in the SGD Algorithm 6 as w ( t+1 ) = w ( t ) - η t [ ∇ f( w ( t ) ) + ξ t ] , (2.3) where ξ t := v t - ∇ f( w ( t ) ) represents the (observed) noise introduced in (2.2) by using a random realization of the exact gradient.

Remark 9. Given T SGD algorithm iterations, the output of SGD can be (but not a exclusively),
  1. the average (after discarding the first few iterations of w ( t ) for stability reasons) w SGD ( T ) = 1 T ∑ t=1 T w ( t ) (2.4)
  2. or the best value discovered w SGD ( T ) = arg min ∀ w t ( f( w ( t ) ) )
  3. or the last value discovered w SGD ( T ) = w ( T )

Note 10. SGD output converges to a local minimum, w SGD ( T ) → w * (in some sense), under different sets of regularity conditions. Section 2.2 has a brief analysis. To achieve this, Conditions 11 on the learning rate are inevitable and should be satisfied.

Condition 11. Regarding the learning rate (or gain) { η t } should satisfy conditions
  1. η t ≥ 0 ,
  2. ∑ t=1 ∞ η t = ∞
  3. ∑ t=1 ∞ η t 2 < ∞

Remark 12. The popular learning rates { η t } in Remark rem:The-parameter- in Handout 2, satisfy Condition 11.

Remark 13. Intuition on Condition 11. Assume that v t is bounded. Condition 11((3.)) aims at reducing the effect of the stochasticity in v t (introduced noise ξ t ) because it implies η t ssearrow 0 as t → ∞ and hence allows the chain to converge as w ( t+1 ) - w ( t ) =- η t v t → 0. Condition 11(2.) prevents η t from reducing too fast and allows the generated chain { w ( t ) } to be able to converge. E.g., after t iterations ∥ w ( t ) - w * ∥ =∥ w ( t ) ± w ( 0 ) - w * ∥ ≥ ∥ w ( 0 ) - w * ∥ -∥ w ( t ) - w ( 0 ) ∥ ≥ ∥ w ( 0 ) - w * ∥ - ∑ t=0 ∞ ∥ w ( t+1 ) - w ( t ) ∥ =∥ w ( 0 ) - w * ∥ - ∑ t=0 T-1 ∥ η t v t ∥ However if it was ∑ t=1 ∞ η t < ∞ it would be ∑ t=0 ∞ ∥ η t v t ∥ < ∞ and hence w ( t ) will never converge to w * if the seed w ( 0 ) is far enough from w * .

2.3 Implementation in the learning Problem 1

Proposition 18. Consider that ℓ is non-differentiable. For a randomly drawn z ∼ g( ⋅ ) , the sub-gradient v of ℓ ( w,z ) at point w is an unbiased estimator of the sub-gradient of the risk R g ( w ) at point w .
Proof. Let v be a sub-gradient of ℓ ( w,z ) at point w , then ℓ ( u,z ) - ℓ ( w,z ) ≥ ⟨ u-w,v ⟩ (2.10) It is R g ( u ) - R g ( w ) = E z ∼ g ( ℓ ( u,z ) - ℓ ( w,z ) |w ) ≥ E z ∼ g ( ⟨ u-w,v ⟩ |w ) = ⟨ u-w, E z ∼ g ( v|w ) ⟩ Hence the sub-gradient of ℓ ( w,z ) is an unbiased estimator of the sub-gradient of R g ( w ) .

Algorithm 20. Stochastic Gradient Descent (SGD) with learning rate η t >0 for Problem 1.
For t=1,2,3,... iterate:
  1. sample z ( t ) ∼ g , and compute v t such that v t ∈ ∂ ℓ ( w ( t ) , z ( t ) )
  2. compute w ( t+1 ) = w ( t ) - η t v t , (2.11)
  3. terminate if a termination criterion is satisfied, e.g. If t ≥ T then STOP

Remark 21. In Algorithm 20 and step 1. sampling z ( t ) ∼ g can be performed either by directly sampling from g (when possible), or by sub-sampling (with or without replacement) from the given dataset { z i ;i=1,...,n } .
