1 Intro and motivation

Note 1. Support Vector Machines (SVM) is a ML procedure for learning linear predictors in high-dimensional feature spaces with regards the sample complexity challenges.

Definition 2. Let w ≠ 0 . Hyperplane of space X ⊆ R d is called the sub-set S={ x ∈ R d : ⟨ w,x ⟩ +b=0 } . It separates X in two half-spaces S={ x ∈ R d : ⟨ w,x ⟩ +b>0 } and S={ x ∈ R d : ⟨ w,x ⟩ +b<0 }

Definition 3. Halfspace (hypothesis space) is hypotheses class H designed for binary classification problems, X ⊆ R d and Y ={ -1,+1 } defined as
where b is called bias.

Definition 4. Each h ∈ H has form h w,b ( x ) = sign ( ⟨ w,x ⟩ +b ) , it takes an input in X ⊆ R d and returns an output in Y ={ -1,+1 } . We may refer to it as halfspace ( w,b ) as this is the only parameter need to fully determine it.
Note 5.  Let S= { ( x i , y i ) } i=1 m be a training set of examples with x i ∈ R d the features and y i ∈ { -1,+1 } the labels.

Note 6. The training set S is linearly separable if there exists a halfspace ( w,b ) such that for all i=1,...,n y i = sign ( ⟨ w, x i ⟩ +b ) or equivalently y i ( ⟨ w, x i ⟩ +b ) >0

Note 7. Let the loss be ℓ ( ( w,b ) ,z ) = 1 ( y i ≠ sign ( ⟨ w, x i ⟩ +b ) ) , and hence the Empirical Risk Function be R S ( w,b ) = 1 m ∑ i=1 m ℓ ( ( w,b ) , z i ) . The Empirical Risk Minimisation (ERM) halfspace ( w * , b * ) is ( w * , b * ) = arg min w,b ( R S ( w,b ) ) = arg min w,b ( 1 m ∑ i=1 m ℓ ( ( w,b ) , z i ) )

Definition 9. Margin of a hyper-plane with respect to a training set is defined to be the minimal distance between a point in the training set and the hyper-plane.
Note 10. Support Vector Machines (SVM) aims at learning the maximum margin separating hyperplane. The rational is that if a hyperplane has a large margin, then it will still separate the training set even if we slightly perturb each instance.

2 Hard Support Vector Machine

Note 11. Hard Support Vector Machine (Hard-SVM) is the learning rule in which we return an ERM hyperplane that separates the training set with the largest possible margin.

Algorithm 12.  (Hard-SVM) Given a linearly separable training sample S= { ( x i , y i ) } i=1 m the Hard-SVM rule for the binary classification problem is:
Solve ( w ˜ , b ˜ ) = arg min ( w,b ) ∥ w ∥ 2 2 (2.1) subject to y i ( ⟨ w, x i ⟩ +b ) ≥ 1, ∀ i=1,...,m (2.2)

3 Soft Support Vector Machine

Note 17. Hard-SVM assumes the strong assumption that the training set is linearly separable.

Note 18. Soft Support Vector Machine (Soft-SVM) aims to relax the strong assumption of Hard-SVM that the training set is linearly separable, with purpose to be extend the scope of application. Soft-SVM is given below.

Algorithm 19.  (Soft-SVM) Given a linearly separable training sample S= { ( x i , y i ) } i=1 m the Hard-SVM rule for the binary classification problem is:
Solve ( w * , b * , ξ * ) = arg min ( w,b, ξ ) ( λ ∥ w ∥ 2 2 + 1 m ∑ i=1 m ξ i ) (3.1) subject to y i ( ⟨ w * , x i ⟩ + b * ) ≥ 1- ξ i , ∀ i=1,...,m (3.2) ξ i ≥ 0, ∀ i=1,...,m (3.3)
Note 20. To relax the linearly separable training set assumption, Soft-SVM relies on replacing the “harder” constraint (2.2) with the “softer” one in 3.3 through the introduction of non-negative unknown quantities { ξ i } i=1 m controlling how much the separability assumption (2.2) is violated. Soft-SVM learns all ( w,b, ξ ) via the minimization part in (3.3) where the trade off between the two terms is controlled via the user specified parameter λ .

From 3.3, it is ξ i ≥ 1- y i ( ⟨ w * , x i ⟩ + b * ) , and from 3.3, it is ξ i ≥ 0 . If y i ( ⟨ w, x i ⟩ +b ) ≥ 1 , the best assignment in 3.4 is ξ i =0 because it is ξ i ≥ 0 from 3.3.
