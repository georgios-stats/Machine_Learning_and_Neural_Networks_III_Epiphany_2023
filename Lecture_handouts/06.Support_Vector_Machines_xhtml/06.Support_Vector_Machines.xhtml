1 Intro and motivation

Note 1. Support Vector Machines (SVM) is a ML procedure for learning linear predictors in high-dimensional feature spaces with regards the sample complexity challenges. Due to a duality property, SVM have sparse solutions, so that predictions for new inputs depend only on quantities evaluated at a subset of the training data points.

Definition 2. Let w ≠ 0 . Hyperplane in space X ⊆ R d is called the sub-set S={ x ∈ R d : ⟨ w,x ⟩ +b=0 } . It separates X in two half-spaces S + ={ x ∈ R d : ⟨ w,x ⟩ +b>0 } and S - ={ x ∈ R d : ⟨ w,x ⟩ +b<0 }

Definition 4. Each halfspace hypothesis h ∈ H has form h w,b ( x ) = sign ( ⟨ w,x ⟩ +b ) (1.1) , it takes an input in X ⊆ R d and returns an output in Y ={ -1,+1 } . We may refer to it as halfspace ( w,b ) as this setting determines it.
Note 5.  Let S= { ( x i , y i ) } i=1 m be a training set of examples with x i ∈ R d the features and y i ∈ { -1,+1 } the labels.

Definition 6. The training set S is linearly separable if there exists a halfspace ( w,b ) such that for all i=1,...,n y i = sign ( ⟨ w, x i ⟩ +b ) or equivalently y i ( ⟨ w, x i ⟩ +b ) >0

Note 7. Let the loss be ℓ ( ( w,b ) ,z ) = 1 ( y i ≠ sign ( ⟨ w, x i ⟩ +b ) ) , and hence the Empirical Risk Function be R S ( w,b ) = 1 m ∑ i=1 m ℓ ( ( w,b ) , z i ) . Definition 8. Margin of a hyper-plane with respect to a training set is defined to be the minimal distance between a point in the training set and the hyper-plane.
Note 9. Support Vector Machines (SVM) aims at learning the maximum margin separating hyperplane Figure (1.1; Right). The rational is that if a hyperplane has a large margin, then it will still separate the training set even if we slightly perturb each instance.

3 Soft Support Vector Machine

Note 20. Hard-SVM assumes the strong Assumption 12 that the training set is linearly separable, that might not always the case, and hence there is need to derive a procedure that weakens this assumption.

Note 21. Soft Support Vector Machine (Soft-SVM) aims to relax the strong assumption of Hard-SVM that the training set is linearly separable (2.4) with purpose to be extend the scope of application. Soft-SVM is given below. I.e., Soft-SVM does not assume Assumption 12.

Note 39. Compared to the Primal problem (Algorithm 13), the dual problem (Note 33) is computationally desirable in cases that the training data size m is smaller that the number of the dimentionality of the feature space d ; i.e. d ≫ m . The solution of the Primal quadratic programming problem in d+1 and complexity O( d+1 ) , while the dual quadratic programming problem has m variables and complexity O( m ) .

Note 40. Compared to the Primal problem, the dual problem (Note 33), can provide sparse solutions relying on a small number of support vectors (see 5.9, 5.10, and 5.11).

Definition 43. Convex learning problem is a learning problem ( H , Z , ℓ ) that the hypothesis class H is a convex set, and the loss function ℓ is a convex function for each example z ∈ Z .

Definition 44. Convex-Lipschitz-Bounded Learning Problem ( H , Z , ℓ ) with parameters ρ , and B , is called the learning problem whose the hypothesis class H is a convex set, for all w ∈ H it is ∥ w ∥ ≤ B , and the loss function ℓ ( ⋅ ,z ) is convex and ρ -Lischitz function for all z ∈ Z .

Definition 45. Convex-Smooth-Bounded Learning Problem ( H , Z , ℓ ) with parameters β , and B , is called the learning problem whose the hypothesis class H is a convex set, for all w ∈ H it is ∥ w ∥ ≤ B , and the loss function ℓ ( ⋅ ,z ) is convex, nonnegative, and β -smooth function for all z ∈ Z .
