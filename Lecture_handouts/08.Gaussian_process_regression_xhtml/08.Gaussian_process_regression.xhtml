2 The Gaussian process regression model

Definition 6. Gaussian process (GP) is a collection of random variables {f( x ) ;x ∈ X } , indexed by label x , where any finite collection of those variables has a multivariate normal distribution. It is fully specified by its mean and covariance functions. It is denotes as f( ⋅ ) ∼ GP ( μ ( ⋅ ) ,C( ⋅ , ⋅ ) ) with mean μ ( x ) := E ( f( x ) ) ,x ∈ X and covariance function C( x,x' ) : = Cov ( f( x ) ,f( x' ) ) ,x,x' ∈ X

Note 7. Essentially, GP is a distribution defined over functions.

Note 8. Consider a function η : X → R with η ( x ) = ⟨ ψ ( x ) ,w ⟩ where ψ ( x ) is a vector of known bases (feature) functions mapping from the input space tot he feature space F , and w ∈ R d is an unknown vector a priori following a normal distribution w ∼ N ( 0,V ) , where the prior mean is set to zero denoting complete uncertainty about the sign of w 's. Then the marginal η ( ⋅ ) follows a Normal distribution as a linear transformation of Normal variates with mean E ( η ( x ) ) =0 and covariance Cov ( η ( x ) , η ( x' ) ) = ψ ( x ) ⊤ V ψ ( x' ) for any x,x' ∈ X . Based on the Kernel trick and Definition 6, we can equivalently specify η ( ⋅ ) ∼ GP ( 0,C( ⋅ , ⋅ ) ) for some kernel / covariance function C( x,x' ) = ψ ( x ) ⊤ V ψ ( x' ) .

Definition 16. A d -dimensional random variable x ∈ R d is said to have a multivariate Normal (Gaussian) distribution, if for every d -dimensional fixed vector α ∈ R d , the random variable α ⊤ x has a univariate Normal (Gaussian) distribution.

Definition 17. We denote the d -dimensional Normal distribution with mean μ and covariance matrix Σ ≥ 0 as N d ( μ , Σ ) .

Notation 18. The d -dimensional standardized Normal distribution is N d ( 0,I ) .

Proposition 19. Let random variable x ∼ N d ( μ , Σ ) , fixed vector c ∈ R q and fixed matrix A ∈ R q × R d . The random vector y=c+Ax has distribution y ∼ N q ( c+A μ ,A Σ A ⊤ ) .

Proposition 20.  Let a d -dimensional random vector x ∼ N ( any ) ( μ , Σ ) .
  1. Let y=Ax and z=Bx , where A ∈ R q × d and B ∈ R k × d : The vectors y=Ax and z=Bx are independent if and only if A Σ B ⊤ =0 .
  2. Let x=( x 1 ,..., x d ) ⊤ : The x 1 ,..., x d are mutually independent if and only if the corresponding off diagonal parts of the Σ are zero.

Proposition 21. Any sub-vector of a vector with multivariate Normal distribution has a multivariate Normal distribution.

Proposition 22. [Marginalization & conditioning] Let x ∼ N d ( μ , Σ ) . Consider partition such that x= [ x 1 x 2 ]; μ = [ μ 1 μ 2 ]; Σ = [ Σ 1 Σ 21 ⊤ Σ 21 Σ 2 ], where x 1 ∈ R d 1 , and x 2 ∈ R d 2 Then:
  1. For the marginal, it is x 1 ∼ N d 1 ( μ 1 , Σ 1 ) .
  2. For the conditional, if Σ 1 >0 , it is x 2 | x 1 ∼ N d 2 ( μ 2|1 , Σ 2|1 ) where μ 2|1 = μ 2 + Σ 21 Σ 1 -1 ( x 1 - μ 1 ) and Σ 2|1 = Σ 2 - Σ 21 Σ 1 -1 Σ 21 ⊤ (A.1)
